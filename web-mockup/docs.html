File Structure

Tenax is a subtle and striking admin template to help you set up all the design elements you might need when creating an admin panel.

Important: If you need to add some custom CSS, do not use the style.css file but for instance the yourname-custom.css file (rename, add css, link it) to avoid loosing your customization when upgrading to a newer version of Liberator.

Responsive @media queries

We use @media queries in our style.css file to add specific css in order to make our template responds best to different devices resolution.

A media query consists of a media type and zero or more expressions that check for the conditions of particular media features. Among the media features that can be used in media queries are ‘width’, ‘height’, and ‘color’. By using media queries, presentations can be tailored to a specific range of output devices without changing the content itself.
Those @media queries are in addition to the ones included in bootstrap.css file. They are mostly here to adapt our specific layout and demo content to different screen width.

For example we use it to change the sidebar navigation display .nav.show
{ width: auto; }, and for other uses similar to this.

According to your needs you might want to add new @media queries for your specific content or to override ours. You should not do it in the styles.css file but create a new specific css file and link it to your HTML. See file structure section for more details.
